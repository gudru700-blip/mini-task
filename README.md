# mini-task
1. Код открывает каждый PDF файл в папке и извлекает текст со всех страниц.
2. Код ищет систематические названия с элементами, названия с характерными суффиксами (-amide, -amine, -acid, -one, -ol, -ene, -ane, -ine, -yl, -ate, -ide), тривиальные названия, названия в формате N-...-...-carboxamide и названия с брутто-формулами.
3. Для каждого найденного названия код ищет биоактивности в той же строке и в следующих трех строках.
4. Для каждого соединения формируется запись.
5. Все найденные соединения из всех PDF файлов собираются в общий список. Из общего списка удаляются повторяющиеся соединения по названию.
6. Каждое уникальное соединение проверяется в PubChem. Соединения, которые не найдены в PubChem, удаляются из базы данных.
7. Для каждого прошедшего проверку соединения генерируется уникальный ID (C1, C2, C3...), добавляются синонимы и дата создания.
